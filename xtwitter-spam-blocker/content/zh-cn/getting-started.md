# X/Twitter Spam Blocker 快速开始

X/Twitter Spam Blocker 的使用流程很简单。安装后默认配置即可使用。

## 1. 从 Chrome Web Store 安装

从 Chrome Web Store 安装 X/Twitter Spam Blocker。安装后，扩展图标会出现在浏览器工具栏或扩展菜单中，默认保护设置已经适合正常浏览。

## 2. 打开 X/Twitter 并正常浏览

访问 `https://x.com`，像平时一样使用 X/Twitter。在支持的页面上，扩展会在本地浏览视图中清理命中的垃圾内容。

## 3. 在浮动面板中审核风险账号

当扩展发现命中高风险垃圾信号的账号时，这些账号会出现在 X/Twitter 页面上的浮动审核面板中。

你可以在面板里查看检测到的账号，并决定哪些账号需要拉黑。

## 4. 确认选中的拉黑操作

选择你想拉黑的账号，然后确认操作。扩展只会拉黑你选中并确认的账号。

没有选中的账号不会被拉黑。

## 5. 查看保护概览和活动记录

打开保护概览，可以查看已检查推文、已清理推文、高风险账号和拉黑结果等本地保护统计。

通过活动记录，你可以查看本设备上的隐藏内容、成功拉黑和拉黑失败的尝试。

## 6. 按需更新设置

需要暂停保护、切换界面语言、调整过滤选项、修改本地历史保留时间或清除本地数据时，可以打开设置。

如果隐藏内容影响了浏览，可以在设置中临时关闭过滤功能，被隐藏的内容会重新正常显示。
