# 支持与联系

如果你需要 X/Twitter Spam Blocker 的帮助，可以通过邮件联系我们，加入 Discord 社群，或在 GitHub 上提交 Issue。

## 邮件

隐私问题、支持请求或产品反馈可以发送到：

[vocabey@gmail.com](mailto:vocabey@gmail.com)

## Discord 社群

产品讨论、快速问题和社区反馈可以加入 Discord：

[https://discord.gg/Xja6KRNrG](https://discord.gg/Xja6KRNrG)

## GitHub Issues

如果遇到 bug、功能异常、兼容性问题，或希望提出功能建议，可以提交 Issue：

[https://github.com/vocabey/vocabey.github.io/issues](https://github.com/vocabey/vocabey.github.io/issues)

## 反馈时建议包含

报告问题时，建议包含：

- 扩展版本。
- Chrome 版本。
- 出现问题的 X/Twitter 页面类型，例如时间线、通知、搜索、帖子详情或用户主页。
- 实际发生了什么。
- 你期望发生什么。
- 可选截图，前提是截图不暴露私人信息。

## 请不要分享敏感数据

不要发送密码、私信、认证令牌、私人账号数据，或暴露敏感个人信息的截图。

如果报告涉及某个被拉黑的账号，只提供理解问题所需的信息即可。
