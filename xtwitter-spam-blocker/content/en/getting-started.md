# Getting Started with X/Twitter Spam Blocker

X/Twitter Spam Blocker is designed to work with minimal setup. The default configuration is ready to use after installation.

## 1. Install the Extension

Install X/Twitter Spam Blocker from the Chrome Web Store. After installation, the extension icon appears in your browser toolbar or extensions menu, and the default protection settings are ready for normal browsing.

## 2. Open X/Twitter and Browse Normally

Go to `https://x.com` and use X/Twitter as usual. On supported pages, the extension can clean matching spam before it appears in your local browsing view.

## 3. Review Risky Accounts in the Floating Panel

When the extension finds accounts that match high-risk spam signals, they appear in the floating review panel on X/Twitter.

Use the panel to see the detected accounts and decide which ones should be blocked.

## 4. Confirm Selected Blocks

Select the accounts you want to block, then confirm the action. The extension only blocks accounts you select and confirm.

Accounts you leave unselected are not blocked.

## 5. Check Dashboard and Activity Log

Open the dashboard to see local protection statistics such as checked tweets, cleaned tweets, high-risk accounts, and block results.

Use the Activity Log to review hidden content, successful blocks, and failed block attempts on this device.

## 6. Update Settings When Needed

Open Settings when you want to pause protection, change the interface language, adjust filtering options, change local history retention, or clear local data.

If hidden content affects your browsing, you can temporarily turn off filtering in Settings and the hidden content will appear normally again.
